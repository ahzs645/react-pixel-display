## PixelDisplayProps

All props are optional. The component renders a blank display with default values if no props are provided.

<Demo client:load />

```tsx
<PixelDisplay />
```

### Content

| Prop       | Type         | Default      | Description                                    |
| ---------- | ------------ | ------------ | ---------------------------------------------- |
| `text`     | `string`     | —            | Text to render on the display                  |
| `pixels`   | `HexColor[]` | —            | Raw pixel data (overrides `text` when provided)|

<Demo client:load text="Hello!" foregroundColor="#00ff88" />

### Dimensions

| Prop       | Type     | Default | Description                              |
| ---------- | -------- | ------- | ---------------------------------------- |
| `width`    | `number` | `64`    | Display width in pixels                  |
| `height`   | `number` | `16`    | Display height in pixels                 |
| `scale`    | `number` | `8`     | Visual scale multiplier                  |
| `pixelGap` | `number` | `0.15`  | Gap between pixels (0 = solid)           |

<Demo client:load text="TINY" width={32} height={10} scale={8} foregroundColor="#ff6600" />

<Demo client:load text="WIDE" width={128} height={16} scale={3} foregroundColor="#00ccff" />

### Appearance

| Prop              | Type       | Default     | Description                      |
| ----------------- | ---------- | ----------- | -------------------------------- |
| `foregroundColor`  | `HexColor` | `'#ff6600'` | Text / pixel color               |
| `backgroundColor`  | `HexColor` | `'#111'`    | Background color                 |
| `glow`            | `boolean`  | `true`      | Enable LED glow bloom            |

<Demo client:load text="GLOW" foregroundColor="#ff00ff" backgroundColor="#1a001a" glow={true} />

<Demo client:load text="NO GLOW" foregroundColor="#ff00ff" backgroundColor="#1a001a" glow={false} />

### Effects

| Prop     | Type         | Default   | Description                |
| -------- | ------------ | --------- | -------------------------- |
| `effect` | `EffectName` | `'fixed'` | Animation effect name      |
| `speed`  | `number`     | `50`      | Animation speed            |

<Demo client:load text="BREEZE" effect="breeze" speed={40} foregroundColor="#00ff88" />

See the [Effects guide](/guides/effects/) for all 24 effect names and live demos.

### Font

| Prop           | Type           | Default          | Description                       |
| -------------- | -------------- | ---------------- | --------------------------------- |
| `font`         | `FontType`     | `'legacy'`       | Font renderer: `'legacy'` \| `'canvas'` \| `'bdf'` |
| `fontName`     | `string`       | `'VCR_OSD_MONO'` | Font name for canvas/BDF          |
| `fontResolver` | `FontResolver` | —                | Custom URL resolver for font files|

<Demo client:load text="LEGACY" font="legacy" foregroundColor="#ff6600" />

See the [Fonts guide](/guides/fonts/) for details on each font system.

### Renderer

| Prop       | Type           | Default       | Description                                |
| ---------- | -------------- | ------------- | ------------------------------------------ |
| `renderer` | `RendererType` | `'imagedata'` | Backend: `'imagedata'` \| `'canvas'` \| `'svg'` |

<Demo client:load text="SVG" renderer="svg" foregroundColor="#00ff88" />

See the [Renderers guide](/guides/renderers/) for a comparison.

### Callbacks

| Prop      | Type         | Default | Description                                |
| --------- | ------------ | ------- | ------------------------------------------ |
| `onReady` | `() => void` | —       | Called when the component is ready to render|

### Styling

| Prop        | Type                    | Default | Description                 |
| ----------- | ----------------------- | ------- | --------------------------- |
| `className` | `string`                | —       | CSS class for the outer div |
| `style`     | `React.CSSProperties`  | —       | Inline styles for outer div |
