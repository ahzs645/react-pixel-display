The `effect` prop selects an animation effect. There are three categories: **text**, **ambient**, and **color**. Effects inspired by the [MD_Parola](https://github.com/MajicDesigns/MD_Parola) and [ESP32-HUB75-MatrixPanel-DMA](https://github.com/mrcodetastic/ESP32-HUB75-MatrixPanel-DMA) LED libraries are also available.

## Text effects

Text effects animate your text content. Pass any text via the `text` prop.

### fixed (default)

Static display — no animation.

<Demo client:load text="STATIC" effect="fixed" foregroundColor="#ff6600" />

```tsx
<PixelDisplay text="STATIC" effect="fixed" />
```

### scroll_rtl

Text scrolls continuously from right to left.

## Controlling speed

All animated effects accept a `speed` prop (default `100`). Lower values are slower, higher values are faster:

<Demo client:load text="SLOW" effect="scroll_rtl" speed={15} foregroundColor="#ff6600" />

<Demo client:load text="FAST" effect="scroll_rtl" speed={80} foregroundColor="#ff6600" />

```tsx
<PixelDisplay text="SLOW" effect="scroll_rtl" speed={15} />
<PixelDisplay text="FAST" effect="scroll_rtl" speed={80} />
```
